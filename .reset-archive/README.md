# Archivo de Reset del Sistema de Agentes

Creado: 2026-04-17 durante la reorganización del sistema de agentes de Conniku.

## Propósito

Esta carpeta contiene archivos del sistema viejo que fueron reemplazados,
pero que se conservan como referencia por si se necesita consultar algo
del pasado antes de borrarlos definitivamente.

## Organización

- claude-old/: contenido anterior de .claude/ (settings, etc.)
- root-orphans/: agentes huérfanos y archivos sueltos de la raíz
- previews/: archivos HTML de preview sin uso productivo
- scripts/: scripts de utilidad que ya no se usan
- rescued-content/: contexto rescatado y reorganizado del sistema viejo
- investigation-results/: outputs de investigación sobre archivos ambiguos

## Decisión final

Cuando el sistema nuevo esté estable y verificado (final de Fase 5),
esta carpeta completa se elimina. No antes.
