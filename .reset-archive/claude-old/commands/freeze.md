Congela un archivo o funcion en FROZEN.md para que no pueda ser modificado sin autorizacion explicita de Cristian.

## Instrucciones:
1. Recibe como argumento la ruta del archivo (ej: `src/pages/Login.tsx`)
2. Lee FROZEN.md completo
3. Verifica que el archivo NO este ya en la lista
4. Si ya esta → informar "Este archivo ya esta congelado"
5. Si no esta → agregar una fila nueva a la tabla con:
   - Archivo: la ruta proporcionada
   - Seccion: "completo" (o lo que Cristian especifique)
   - Confirmado: fecha de hoy
   - Nota: pedir a Cristian una nota breve o dejar vacio
6. Mostrar la tabla actualizada a Cristian para confirmar

Argumento: $ARGUMENTS
