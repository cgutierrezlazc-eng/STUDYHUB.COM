# docs/inspecciones/

Documentación de las iteraciones de **Capa 6 — Inspección humana en web online** del protocolo de 7 capas (CLAUDE.md §18.4).

## Convención de nombres

`{bloque}-iter-{N}.md`

Ejemplos:
- `bloque-1-auth-edad-iter-1.md` — primera inspección de Cristian en preview
- `bloque-1-auth-edad-iter-2.md` — tras correcciones, segunda inspección
- `dashboard-profesor-iter-1.md` — inspección inicial del bloque Dashboard Profesor

## Qué contiene cada archivo

- Fecha y hora de la inspección
- URL de preview inspeccionada (Vercel)
- Flujos probados (golden path + edge cases)
- Dispositivos probados (desktop, tablet, mobile)
- Hallazgos: bugs, problemas UX, ajustes de texto, accesibilidad, rendimiento, cambios de alcance
- Decisión final: OK para Capa 7 / retrocede a capa X con correcciones específicas

## Sin iteraciones todavía

Este directorio se llenará a medida que los bloques pasen por Capa 6 del protocolo. A la fecha de creación (2026-04-18) ningún bloque ha llegado a esta capa.
