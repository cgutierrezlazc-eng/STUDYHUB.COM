LOGO.09 RULES.INVIOLABLE
  • u_color == #0D0F10 · always · no exception
  • dot_color == #FF4A1C · always · no exception
  • u-pack_background == #D9FF3A · always · no exception
  • palette_size == 4 · no additional colors
  • u-pack contains u + dot · never separated
  • wordmark rotation == 0deg · no rotation allowed
  • font_family == Funnel Display 900 · no substitution
  • final "." is present · never omitted
  • aspect_ratio preserved · no stretch · no skew
  • wordmark modification requires approval from Cristian Gutiérrez Lazcano
LOGO.10 FORBIDDEN.LIST
  • apply filter: blur / drop-shadow / hue-rotate / invert
  • apply transform: rotate(!=0) / skew / scaleY independent of scaleX
  • apply outline / border around the tile
  • apply glow or bevel
  • substitute font-family
  • change u fill from #0D0F10
  • change dot fill from #FF4A1C
  • change tile fill from #D9FF3A
  • use wordmark with opacity < 30%
  • overlay text on top of the u-pack
  • wrap logo inside additional box / pill / ring
  • place logo within 2x clear_space of another logo without divider
  • animate with geometry distortion (squash / stretch / morph)
  • use logo as decorative typography inside illustrations
  • emit low-resolution raster below min_digital
LOGO.11 VALIDATION.CHECKLIST

Execute before merging any change that touches logo rendering.

  • Render app_icon at 16px · u is readable · dot is visible
  • Render app_icon at 1024px · geometry matches LOGO.04 spec
  • Render wordmark on --paper · connik is #0D0F10
  • Render wordmark on --ink · connik is #F5F4EF · tile and u unchanged
  • Render wordmark on all extended palette colors (pink, cyan, cream, violet) · tile stays #D9FF3A, u stays #0D0F10, dot stays #FF4A1C
  • Wordmark in one line · no wrap · no overflow ellipsis
  • No rotation applied
  • No filter CSS property applied to logo element
  • No shadow / glow / outline / border
  • Clear_space respected · no element inside the defined margin
  • Exported PNG preserves vector geometry · no blur · no rasterization artifacts
  • Font family loaded before logo renders · or <text> converted to <path>
  • WCAG AA contrast between connik and background · ratio >= 4.5
  • WCAG AA contrast between u and tile · ratio >= 4.5 · validated
