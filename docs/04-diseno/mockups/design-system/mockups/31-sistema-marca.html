Un logo. Una historia. Un sistema que no se negocia.

El logo de Conniku vive en web, móvil, tiendas de apps, splash screens, emails, cursos, material impreso y en cualquier lugar donde la marca aparezca. Este documento visual define cómo se construye, qué variantes existen, y dónde se usan. Cualquier desviación requiere aprobación explícita.

v1.0.0 · oficial

El tile, el corazón de la marca

Cuadrado redondeado tinta, panel lima desplazado al centro, recorte circular hacia la derecha y una chispa naranja al centro del recorte. Todo el sistema visual de Conniku nace de esta geometría. Cualquier superficie digital o impresa reutiliza este mismo tile; nunca se redibuja.

08

Correcto / Incorrecto

el sistema no se negocia

Correcto sí

Sobre warm paper, con espacio libre respetado
Sobre tinta, usar variante color o invertida
Sobre paleta extendida, tile a color sin modificar
Monocromo dark cuando el color no aplica

Incorrecto no

No rotar el logo fuera de 0°
No deformar el aspect ratio
No aplicar filtros ni recolorear
No aplicar glows, sombras extrañas ni efectos
