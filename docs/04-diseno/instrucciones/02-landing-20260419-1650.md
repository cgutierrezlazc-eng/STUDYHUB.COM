### Google OAuth B-flow (state machine)

```
STATE: authMode: 'email' | 'google'

INITIAL: authMode = 'email'
  - Muestra todos los campos (flow-email-only visibles)
  - Banner Google oculto
  - Social row visible

ON CLICK btn-google-social:
  1. Trigger OAuth real: supabase.auth.signInWithOAuth({ provider: 'google' })
  2. On success, authMode = 'google'
  3. Pre-fill nombre + email desde Google OAuth payload
  4. Hide .flow-email-only, .social-row, .divider
  5. Show #google-banner with user.email
  6. Update step-label: "Con Google · confirma tu edad y acepta"

ON CLICK btn-google-change (from banner):
  1. authMode = 'email'
  2. Clear session (no logout · solo estado local)
  3. Show .flow-email-only, .social-row, .divider
  4. Hide #google-banner
  5. Restore step-label: "Paso 1 de 3 · datos básicos"

SUBMIT (shared email+google):
  - Validate:
    * birthdate >= 18 años (backend recalcula)
    * declaration checkbox checked
    * Si email-mode: email format, password ≥ 8, passwords match
  - POST /api/v1/auth/register  OR  OAuth callback finalize
  - On success, navigate('/verify')
```

### OTP state machine

```
STATE:
  codeSent: boolean = false
  otpValues: string[6] = ['','','','','','']
  timerSeconds: number = 0
  timerInterval: NodeJS.Timer | null

ON CLICK btn-send-code:
  IF !email.includes('@') → focus email, return
  ELSE:
    1. POST /api/v1/verify/email/send-code  body: { email, university }
    2. codeSent = true
    3. Enable all 6 otp-input (remove disabled)
    4. btn-send-code.textContent = "Código enviado ✓" · add class .sent
    5. Enable btn-resend-code
    6. startTimer(600 seconds)  // 10 min
    7. Auto-focus otpInputs[0]

ON TIMER tick (1s):
  remaining--
  render "Código expira en M:SS"
  IF remaining <= 0:
    clearInterval
    textContent = "Código expirado · reenvía"

ON CLICK btn-resend-code:
  1. POST /api/v1/verify/email/resend  body: { email }
  2. Clear all 6 otpValues, remove .filled class
  3. focus otpInputs[0]
  4. startTimer(600)

ON INPUT otp-input[i]:
  value = value.replace(/\D/g, '')  # only digits
  IF value:
    add class .filled
    IF i < 5: focus otpInputs[i+1]
  ELSE:
    remove class .filled

ON KEYDOWN otp-input[i]:
  Backspace + empty  + i>0   → focus otpInputs[i-1]
  ArrowLeft  + i>0            → focus otpInputs[i-1]
  ArrowRight + i<5            → focus otpInputs[i+1]

ON PASTE otp-input[any]:
  preventDefault
  pasted = clipboardData.replace(/\D/g, '').slice(0, 6)
  fill otpInputs[0..pasted.length]
  focus otpInputs[min(pasted.length, 5)]

SUBMIT (btn-verify):
  code = otpValues.join('')
  POST /api/v1/verify/email/confirm  body: { email, code }
  On success → navigate('/onboarding/carrera')
```
