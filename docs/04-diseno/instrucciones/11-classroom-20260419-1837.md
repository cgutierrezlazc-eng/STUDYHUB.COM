## MODULE.11.04 · STATE MACHINE (alto nivel)

```
PREREQ: user authenticated + active class_id reservation

STATE:
  classPhase: 'waiting' | 'live' | 'exercise' | 'ending'
  micEnabled: boolean
  cameraEnabled: boolean
  screenShare: boolean
  handUp: boolean
  planStep: 1..5
  transcript: TranscriptEntry[]
  chatMessages: ChatMessage[]
  recording: { active: boolean, durationSec: number }
  cost: { amountCLP: number, ratePerMin: number }

ON JOIN:
  1. Connect to streaming provider (TBD)
  2. Open WS_CHANNEL for class control events
  3. planStep = 1 · classPhase = 'waiting'
  4. Tutor starts stream → classPhase = 'live'

ON EXERCISE TRIGGER (by tutor):
  classPhase = 'exercise'
  Show exercise-banner with countdown timer
  Student completes · sends answer via chat or transcript

ON PLAN ADVANCE (tutor marks step done):
  planStep++
  Update Row 2 progress bar
  IF planStep > 5: classPhase = 'ending'

ON TERMINAR CLASE (Row 1 button):
  1. Modal de confirmación "¿Cerrar la clase y liberar el pago?"
  2. Stream stopped
  3. WS disconnect
  4. Backend: mark class_id complete · release escrow to tutor
  5. Navigate to /clases/:classId/resumen (vista post-clase con transcript descargable)
```
