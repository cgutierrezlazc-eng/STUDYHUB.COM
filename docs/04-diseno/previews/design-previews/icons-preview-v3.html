Concepto: Relieve Negativo vs Positivo

🔳 Relieve Negativo (inactivo) Ícono blanco recortado sobre un badge de color. El color identifica la sección. Cada ícono tiene su propio color sobrio.
— Paleta muted: índigo, violeta, ámbar, teal, rosa, coral, azul, verde
🟦 Relieve Positivo (activo) El ítem activo toma el pill azul sólido de estilo J con glow. El badge se vuelve translúcido. El texto se vuelve blanco.
— Azul #2D62C8 → #4A90E8 con sombra 30% opacity
Variantes K = cuadrado 30px · L = círculo 30px · M = compacto 26px · N = dark sidebar
