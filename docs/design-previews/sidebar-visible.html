Pregunta: ¿cómo organizar el sidebar cuando todo está visible?

A — Con toggle (referencia)Secciones colapsables. El usuario decide qué ver. Más largo cuando todo abierto.
B — Labels estáticosTodas las secciones siempre visibles con sus labels. El sidebar tiene scroll si el contenido es largo.
C — Separadores visualesSin texto de sección — solo una línea entre grupos. Más limpio, pero menos orientación.
D — Chips de categoríaChips arriba filtran qué sección ves. Sidebar compacto, solo muestra una sección a la vez.
